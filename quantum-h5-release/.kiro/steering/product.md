# Quantum H5

Quantum is an AI-powered assistant platform developed by Lenovo. The project consists of multiple frontend applications (Pods) that provide different AI interaction experiences, along with shared component libraries and services.

## Core Purpose
- AI chat interfaces with streaming responses
- Multi-modal interactions (text, voice, file uploads)
- Theming support (light/dark modes)
- Internationalization (i18n)

## Key Applications
- **Pod1**: Main AI assistant application with full chat capabilities
- **DevUI**: Component development and testing environment
- **quantum-electron-screenshot**: Electron-based screenshot tool
- **Various Pods**: Specialized AI interaction interfaces (Pod0, Pod9, Pod12, etc.)

## Target Platform
- Web (H5) applications
- Electron desktop applications
- Lenovo AI ecosystem integration
